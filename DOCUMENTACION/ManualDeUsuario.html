Manual de Usuario - CoffeeShops

Introducción

Bienvenido a CoffeeShops, la aplicación que te permite explorar y calificar tus cafeterías favoritas. Con CoffeeShops, puedes descubrir nuevas cafeterías, ver sus detalles y compartir tus opiniones.

Funcionalidades Principales

  • Explorar Cafeterías: Navega por una lista de cafeterías con sus nombres, direcciones e imágenes.
  • Ver Detalles: Toca una cafetería para ver su nombre en detalle.
  • Calificar Cafeterías: Asigna una calificación a cada cafetería usando las estrellas de calificación.

Cómo Utilizar la Aplicación

Explorar Cafeterías

Al abrir la aplicación, verás una lista de cafeterías. Puedes desplazarte hacia arriba y hacia abajo para ver todas las opciones.

Ver Detalles

Para ver el nombre de una cafetería en detalle, simplemente toca el elemento de la cafetería en la lista. Se abrirá una nueva pantalla con el nombre de la cafetería.

Calificar Cafeterías

En la lista principal, cada cafetería tiene un sistema de calificación de estrellas. Toca las estrellas para asignar una calificación a la cafetería. Tu calificación se guardará automáticamente.

Accesibilidad

CoffeeShops se ha diseñado pensando en la accesibilidad:

  • Contraste de Colores: Los textos y elementos visuales tienen un contraste adecuado para facilitar la lectura.
  • Interacciones Claras: Los elementos interactivos son fáciles de identificar y usar.
  • Soporte para Lectores de Pantalla: La aplicación es compatible con lectores de pantalla para usuarios con discapacidad visual.
  • Tamaño de texto adaptable: El tamaño del texto se adapta a las configuraciones del dispositivo.

Pruebas de Usabilidad y Accesibilidad

Se realizaron pruebas exhaustivas para asegurar que CoffeeShops sea fácil de usar y accesible:

  • Se verificó que la navegación por la lista de cafeterías es intuitiva.
  • Se comprobó que la calificación de cafeterías funciona correctamente.
  • Se validó que el contraste de colores y el tamaño de los textos son adecuados.
  • Se probó la compatibilidad con lectores de pantalla.
