Tutorial: Explorar Cafeterías

Paso 1: Abrir la Aplicación

Abre la aplicación CoffeeShops en tu dispositivo.

Paso 2: Ver la Lista de Cafeterías

Verás una lista de cafeterías con sus nombres, direcciones e imágenes.

Paso 3: Desplazarse por la Lista

Desliza hacia arriba o hacia abajo para ver todas las cafeterías disponibles.

Paso 4: Seleccionar una Cafetería

Toca el elemento de una cafetería para ver las diferentes opiniones

Paso 5: Regresar a la Lista

Presiona el botón de retroceso para volver a la lista de cafeterías.

¡Listo!

Ahora sabes cómo explorar las cafeterías en CoffeeShops.
